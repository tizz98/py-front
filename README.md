# py-front

A python API wrapper around [Front](https://frontapp.com).

# Installation

```bash
pip install py-front
```

# Usage

## Set api key

```python
import front
front.set_api_key("jwt_token")
```

## Use the api

```python
import front

for conv in front.Conversation.objects.all():
    print(conv.id, conv.subject)
```
